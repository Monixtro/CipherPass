CipherPass is a lightweight password security toolkit designed to help users
evaluate password strength and generate secure passwords.

## Features
- Password strength analysis
- Secure password generator
- Clean, modern UI
- Client-side only (no data stored)

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Project Purpose
This project was built to strengthen practical cybersecurity skills and
demonstrate front-end security concepts.
