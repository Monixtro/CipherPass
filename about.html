About CipherPass

CipherPass is a client-side password security tool designed to help users understand, generate, and evaluate strong passwords using modern security principles.


What CipherPass Does

CipherPass provides two main features:

  • Password Strength Checker: analyzes passwords using entropy, pattern detection, and real-world attack models.
  • Password Generator: creates cryptographically strong passwords locally in your browser.

How Strength Is Measured

Password strength is calculated using Shannon entropy based on character diversity and length, combined with structural penalties for:

  • Common dictionary words and leaked passwords
  • Keyboard patterns and sequences
  • Repeated characters and predictable substitutions
  • Passphrases composed solely of lowercase words

Crack time estimates simulate multiple attacker models, ranging from online rate-limited attacks to nation-state-level offline attacks.


Security & Privacy

CipherPass runs entirely in your browser. No passwords are transmitted, stored, logged, or sent to any server.

All calculations happen locally using JavaScript. Even compromised password detection is performed against a locally loaded wordlist.


Educational Purpose

CipherPass is intended as an educational and practical tool. Crack time estimates are approximations designed to help users make better security decisions, not absolute guarantees.

Real-world security depends on many factors, including hashing algorithms, salting, and rate-limiting protections.
